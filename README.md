# Handwritten PDF to Excel AI

A premium, production-ready SaaS experience for schools and colleges to upload handwritten PDF records and prepare them for Excel export.

## Features

- Modern landing experience with premium animations
- Upload workflow for PDF files up to 25MB
- Dashboard, history, settings, login, and register pages
- Placeholder API routes for upload, extraction, and export
- SEO metadata, OpenGraph, Twitter cards, robots, and sitemap

## Getting started

```bash
npm install
npm run dev
```

## API placeholders

The app includes placeholder endpoints at:

- /api/upload
- /api/extract
- /api/export

These routes currently return stub responses and are ready for future integration with the Python backend.

## Build

```bash
npm run build
```
